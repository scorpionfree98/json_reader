## v0.1.20 更新内容

### 🐛 Bug 修复

1. **分屏模式转义功能修复**
   - 修复分屏模式下转义按钮不生效的问题
   - 统一了 `isExplainEnabled` 状态管理
   - 转义按钮现在能正确切换并影响树视图显示

2. **搜索功能改进**
   - 修复普通文本搜索只找到第一个匹配的问题
   - 改进搜索算法，现在能找到所有匹配项
   - 支持前进/后退导航所有匹配结果

3. **分屏模式复制提示修复**
   - 修复分屏模式下双击复制无提示的问题
   - 优化 CSP（Content Security Policy）配置
   - 添加自定义通知作为 LayUI 的 fallback
   - 复制成功后显示明确的提示信息

### ✨ 新功能

4. **切换视图自动复制**
   - 编辑器模式 ↔ 分屏模式切换时自动复制内容到剪贴板
   - 格式化后的 JSON 内容会自动复制
   - 显示"输出内容已复制到剪贴板"提示

5. **内容保留优化**
   - 切换视图模式时保留所有已输入和格式化的内容
   - 两个模式之间的内容实时同步

### 🔧 技术改进

- 优化 Tauri CSP 配置，允许 LayUI 正常工作
- 添加 `'unsafe-eval'` 到 script-src（解决 LayUI EvalError）
- 添加 CSS class 方式的通知系统（避免 CSP 内联样式限制）
- 改进剪贴板 API 的兼容性（Tauri + 浏览器双重支持）

### 📝 测试

- 新增 4 个 E2E 测试用例
- 所有 46 个测试全部通过
- 覆盖新功能和 Bug 修复的所有场景
