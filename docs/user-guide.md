# JSONFormatter 用户指南

## 1. 简介

JSONFormatter 是一款跨平台的桌面 JSON 格式化工具,提供以下核心功能:

- **JSON 格式化与验证** — 自动格式化 JSON 数据,实时验证语法错误
- **树形视图** — 以可折叠的树状结构展示 JSON 层级关系
- **LaTeX 公式渲染** — 自动识别并渲染 JSON 中的 LaTeX 数学公式
- **多种路径复制格式** — 支持 JSONPath、Python、点号、括号等 6 种路径格式
- **双视图模式** — 编辑器模式和分屏模式自由切换
- **主题切换** — 支持深色/浅色主题
- **剪贴板集成** — 快速粘贴和复制 JSON 数据
- **自动更新** — 自动检测并安装新版本

## 2. 系统要求

### Windows
- Windows 10 或更高版本
- 64 位操作系统
- 可选:WebView2 运行时(部分安装包已内置)

### macOS
- macOS 10.15 (Catalina) 或更高版本
- 支持 Intel 和 Apple Silicon (M1/M2/M3) 芯片

## 3. 安装

### Windows 安装

**方式一:MSI 安装包**
1. 下载 `JSONFormatter_x.x.x_x64_en-US.msi`
2. 双击运行安装程序
3. 按照向导完成安装

**方式二:NSIS 安装包**
1. 下载 `JSONFormatter_x.x.x_x64-setup.exe`
2. 双击运行安装程序
3. 选择安装路径并完成安装

**注意事项:**
- 如遇到 Windows SmartScreen 警告,点击"更多信息"→"仍要运行"
- 部分安装包已内置 WebView2,无需额外安装

### macOS 安装

1. 下载对应芯片的 DMG 文件:
   - Intel 芯片:`JSONFormatter_x.x.x_x64.dmg`
   - Apple Silicon:`JSONFormatter_x.x.x_aarch64.dmg`
2. 打开 DMG 文件
3. 将 JSONFormatter 拖拽到"应用程序"文件夹
4. 首次运行时,右键点击应用 → 选择"打开"以绕过 Gatekeeper

## 4. 功能说明

### 4.1 编辑器模式

编辑器模式提供单一文本编辑区域,适合快速格式化和编辑 JSON。

**主要操作:**

| 操作 | 说明 |
|------|------|
| 格式化 | 点击"格式化"按钮,自动格式化并高亮 JSON |
| 清空 | 点击"清空"按钮,清除编辑器内容 |
| 粘贴 | 点击"粘贴"按钮,从剪贴板粘贴内容 |
| 置顶 | 点击图钉图标,将窗口固定在最前方 |
| 转义 | 切换 JSON 字符串的转义/反转义显示 |
| 主题 | 点击主题图标,在深色/浅色主题间切换 |

### 4.2 分屏模式

分屏模式将界面分为左侧编辑区和右侧树形视图区,适合浏览复杂的 JSON 结构。

**切换方式:** 点击工具栏中的视图切换按钮,在编辑器模式和分屏模式之间切换。

**分屏模式操作:**
- 在左侧编辑区输入或粘贴 JSON
- 点击"格式化"后,右侧自动显示树形视图
- 左右两侧可独立滚动

### 4.3 树形视图

树形视图以层级结构展示 JSON 数据,方便浏览嵌套内容。

**操作说明:**
- **展开/折叠** — 点击节点左侧的箭头图标展开或折叠子节点
- **全部展开** — 点击"展开全部"按钮展开所有节点
- **全部折叠** — 点击"折叠全部"按钮折叠所有节点
- **双击复制** — 双击节点的键名或值,自动按当前路径格式复制路径到剪贴板

**节点类型显示:**
- 字符串值显示为绿色
- 数字值显示为蓝色
- 布尔值显示为橙色
- null 值显示为灰色
- 对象和数组显示子节点数量

### 4.4 LaTeX 公式渲染

JSONFormatter 支持自动识别并渲染 JSON 字符串中的 LaTeX 数学公式。

**语法格式:**
- **行内公式** — 使用 `$...$` 包裹,例如:`"formula": "$E=mc^2$"`
- **块级公式** — 使用 `$$...$$` 包裹,例如:`"formula": "$$\\int_0^\\infty e^{-x} dx = 1$$"`

**示例 JSON:**
```json
{
  "inline": "$a^2 + b^2 = c^2$",
  "block": "$$\\sum_{i=1}^{n} i = \\frac{n(n+1)}{2}$$"
}
```

### 4.5 路径复制格式

在树形视图中双击节点时,可按以下格式复制路径。通过工具栏的格式选择器切换当前格式。

| 格式 | 示例 | 说明 |
|------|------|------|
| 默认 | `data.users[0].name` | 混合点号和括号 |
| 点号 | `data.users.0.name` | 全点号分隔 |
| JSONPath | `$.data.users[0].name` | 标准 JSONPath 格式 |
| 括号 | `["data"]["users"][0]["name"]` | 全括号格式 |
| Python | `data.get("users", [])[0].get("name")` | Python 字典访问格式 |
| 自定义 | 用户自定义模板 | 支持 `{key}` 和 `{index}` 占位符 |

### 4.6 错误提示

当输入的 JSON 存在语法错误时,应用会显示详细的错误信息:

- **行号和列号** — 精确定位错误位置
- **错误上下文** — 显示错误附近的代码片段
- **错误描述** — 说明具体的语法错误类型

### 4.7 自动更新

应用启动时会自动检查更新(可在设置中禁用)。

**更新流程:**
1. 应用检测到新版本后弹出更新提示
2. 点击"立即更新"开始下载
3. 下载完成后按提示重启应用

**禁用自动更新:** 在托盘菜单中选择"禁用更新检查"。

### 4.8 托盘菜单

应用最小化后会在系统托盘显示图标。右键点击托盘图标可访问以下功能:

| 菜单项 | 功能 |
|--------|------|
| 显示/隐藏 | 切换主窗口的显示状态 |
| 置顶 | 切换窗口置顶状态 |
| 开机自启 | 切换开机自动启动 |
| 禁用更新检查 | 关闭自动更新检测 |
| 检查更新 | 手动触发更新检查 |
| 退出 | 完全退出应用 |

## 5. 常见问题

### macOS Gatekeeper 阻止运行

**问题:** 首次打开时提示"无法打开,因为无法验证开发者"

**解决方法:**
1. 在 Finder 中找到 JSONFormatter
2. 按住 Control 键并点击应用图标
3. 选择"打开"
4. 在弹出的对话框中再次点击"打开"

或者通过终端运行:
```bash
xattr -cr /Applications/JSONFormatter.app
```

### Windows SmartScreen 警告

**问题:** 安装时出现"Windows 已保护你的电脑"提示

**解决方法:**
1. 点击"更多信息"
2. 点击"仍要运行"

### 如何禁用自动更新

1. 右键点击系统托盘中的 JSONFormatter 图标
2. 选择"禁用更新检查"
3. 该设置会持久保存,重启后仍然有效

### 应用关闭后在哪里找到

点击窗口关闭按钮后,应用会最小化到系统托盘而非完全退出。在系统托盘找到 JSONFormatter 图标,双击或右键选择"显示"即可恢复窗口。
